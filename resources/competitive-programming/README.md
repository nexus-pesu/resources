# Competitive Programming

### Sites to Practice/Compete

There are many sites where you can practice and develop your CP skills. Here are some of the most well-known practicing and competing sites:

- [Codeforces](https://codeforces.com/)
- [AtCoder](https://atcoder.jp/)
- [CodeChef](https://www.codechef.com/)
- [CSES](https://cses.fi/)
- [LeetCode](https://leetcode.com/)
- [Binary Search](https://binarysearch.com/)
- [HackerRank](https://www.hackerrank.com/)
- [HackerEarth](https://www.hackerearth.com/)
- [SpOJ](https://www.spoj.com/)
- [Google's Coding Competitions](https://codingcompetitions.withgoogle.com/)
- [CS Academy](https://csacademy.com/)
- [Kattis](https://open.kattis.com/)
- [TopCoder](https://www.topcoder.com/)
- [CodeWars](https://www.codewars.com/)
- [Project Euler](https://projecteuler.net/)
- [Timus](https://acm.timus.ru/)
- [POJ](http://poj.org/)
- [Toki](https://tlx.toki.id/)
- [USACO](http://usaco.org/)
- [Library Checker](https://judge.yosupo.jp/)
- [CodinGame](https://www.codingame.com/)

These sites provide world-class resources for training and any hardworking individual can make it big with continuous practice and effort. To stay up-to-date about upcoming contests, one can use [this](https://clist.by/) site.
